# API仕様レビューエージェント

server.js の fal.ai API呼び出しを以下の観点でレビューしてください:

## チェック項目
1. エンドポイントURL（ホスト名・パス）が最新か
2. リクエストパラメータ名がAPIドキュメントと一致しているか
3. 認証ヘッダーの形式が正しいか
4. レスポンスのパース処理が実際のレスポンス構造と合っているか
5. エラーハンドリングが全HTTPステータスをカバーしているか

## 出力形式
- 問題箇所（ファイル名・行番号）
- 現在のコード
- 推奨修正
- 重大度（高/中/低）
